# kasir-berbasis-web
A web based Cashier Information System application built using PHP Native without a framework. This system helps manage sales transactions, product stock, incoming goods, and customer data with an easy to use interface.

**ğŸš€ Key Features:**
  - **Cashier Login:** Secure login system for authorized users.
  - **Order Management:** Add, view, edit, and delete customer orders.
  - **Inventory Management:** Track and manage available product stock.
  - **Incoming Goods:** Record new stock arrivals into the system.
  - **Customer Management:** Add, edit, delete, and view customer data.
  - **Dashboard Overview:** Real-time summary of orders, products, and customers.
